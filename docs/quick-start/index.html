Get up and running with Internet Speed Monitor in just a few minutes!

Step 1: Download the Application

Choose the appropriate version for your operating system:

  • macOS: Download the .dmg file from the releases page
  • Windows: Download the .exe installer
  • Linux: Download the .AppImage file
üí° Tip: Always download from the official GitHub releases page to ensure you get the latest, secure version.

Step 2: Install the Application

macOS Installation:

  1. Double-click the downloaded .dmg file
  2. Drag the Internet Speed Monitor app to your Applications folder
  3. Launch from Applications or Spotlight search

Windows Installation:

  1. Double-click the downloaded .exe file
  2. Follow the installation wizard
  3. Launch from Start Menu or Desktop shortcut

Linux Installation:

  1. Make the AppImage executable: chmod +x Internet-Speed-Monitor.AppImage
  2. Double-click to run, or execute from terminal
‚ö†Ô∏è Security Note: On macOS, you may need to allow the app in System Preferences > Security & Privacy if you see a security warning.

Step 3: First Launch

When you first open Internet Speed Monitor, you'll see:

  • Current Speed section: Shows real-time speed measurements
  • Speed History chart: Visual representation of your connection over time
  • Monitoring Controls: Set your testing preferences
üìä Data Storage: All your speed test data is stored locally on your device. No data is sent to external servers.

Step 4: Configure Monitoring Settings

Set up automatic monitoring to track your internet performance:

Test Once Now:

  • Click "Test Once Now" for an immediate speed test
  • Get instant results without starting automatic monitoring
  • Perfect for quick checks or troubleshooting

Choose Test Interval:

  • Click the interval dropdown (default: 5 minutes)
  • Select from 1-60 minutes based on your needs
  • Shorter intervals = more data points, but more bandwidth usage

Start Monitoring:

  1. Click the "Start Monitoring" button
  2. The app will begin running speed tests automatically
  3. Watch the status indicator change to "Running" during tests and "Sleeping" between tests
  4. See countdown timer showing when your next test will run
  5. Speed data will appear in real-time on the dashboard
üîã Performance Tip: For continuous monitoring, consider starting with 15-30 minute intervals to balance data collection with system resources.

Step 5: Understanding Your Data

Real-time Information:

  • Status Indicator: Shows current state (Stopped/Running/Sleeping)
  • Test Running: Visual indicator when speed tests are actively running
  • Next Test: Countdown timer to next scheduled test
  • Current Stats: Latest download/upload speeds and ping
  • Median Stats: Median values across all your historical data

Speed Metrics:

  • Download Speed: How fast you can receive data (Mbps)
  • Upload Speed: How fast you can send data (Mbps)
  • Ping: Network latency/response time (milliseconds)

Reading the Chart:

  • Blue line shows download speeds over time
  • Green line shows upload speeds over time
  • Hover over points to see exact values and timestamps

Step 6: Export Your Data

Save your speed test results for further analysis:

  1. Click the "Export CSV" button
  2. Choose a location to save the file
  3. Open in Excel, Google Sheets, or any spreadsheet application

The exported CSV contains:

  • Timestamp of each test
  • Download speed (Mbps)
  • Upload speed (Mbps)
  • Ping time (ms)
